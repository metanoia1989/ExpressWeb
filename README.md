# 开始学习JS技术栈
Node, React, TypeScript, MongoDB 这些。

## Express 学习要点
1. 表单处理 文件上传
2. csrf 防跨域请求伪造攻击 cors 跨站资源共享
3. 路由架构 模块化
4. 模板引擎 Pug
5. 数据持久化 文件读写 数据库使用
6. 认证 注册登录 权限系统
7. 常用第三方中间件
8. 有类型系统的TypeScript
